Automation Framework
This framework helps the tester in automating test execution of AUT by writing the selenium scripts.


How to use
1. Download the zip file / clone the project into your local computer
2. Extract the zip file
3. Import the project into any of the IDE available like Eclipse / Intellij IDEA
4. Make sure TestNG is installed in the IDE
5. Clean the Project and install any missing dependency by running the maven command - mvn clean install
6. Go to config.properties file and set the browser in which you want to execute the tests (Firefox / Chrome)
6. Now right click on the testng.xml file and run as TestNG
7. Test Execution will be started
8. At the end of the Execution, Extent reports will be generated under the HTML Reports folder
